# lun4by

Xây dựng hệ sinh thái **Lunaby** - AI products, Discord experiences, SDKs, và công cụ canvas.

## About

**lun4by** là organization tập trung vào các sản phẩm xoay quanh **Lunaby**:

- **Lunaby**: Discord AI bot hỗ trợ trò chuyện, tạo ảnh, moderation utilities, XP/profile system và nhiều tính năng cộng đồng.
- **lunaby-sdk**: SDK TypeScript/JavaScript chính thức để tích hợp Lunaby API vào ứng dụng riêng.
- **Lunaby-canvas**: thư viện canvas/card generator cho Discord: rank card, welcome/leave, profile, captcha, social-style card và nhiều layout khác.
- **lunaby-docs**: tài liệu cho hệ sinh thái Lunaby.

## Core Projects

### Lunaby
Discord bot AI với định hướng production, hỗ trợ:
- AI chat
- image generation
- command system theo module
- role/quota/cooldown management
- guild-level settings
- XP, profile, moderation và utility features

### lunaby-sdk
SDK chính thức cho JavaScript/TypeScript:
- chat completions
- streaming responses
- image generation
- type-safe API integration
- OpenAI-style developer experience

### Lunaby-canvas
Thư viện render hình ảnh/card dành cho Discord bots:
- Rank card
- Level up card
- Welcome/Leave
- Profile card
- Captcha
- Social mockups như Tweet/Instagram
- Utility image effects

## Tech Stack

- **Node.js**
- **TypeScript / JavaScript**
- **discord.js**
- **MongoDB**
- **MariaDB**
- **Canvas / @napi-rs/canvas**

## Vision

Mục tiêu của `lun4by` là xây dựng một hệ sinh thái AI mang tính ứng dụng cao:
- dễ tích hợp cho developer
- tối ưu cho cộng đồng Discord
- có trải nghiệm sản phẩm riêng, không chỉ là wrapper API
- mở rộng dần sang SDK, docs, canvas engine và backend services
